      3. 六枚ろくまい落おちのポイント

      八枚はちまい落おちのように棒銀ぼうぎんで崩くずすことは難むずかしくなります。 しかし上手うわて陣じんは両端りょうはしの守まもり駒こまが少すくないので、左右さゆうどちらかの端はし攻せめで崩くずせます (左右図さゆうず)。 端はしから優勢ゆうせいを作つくり、勝かちにつなげる力ちからが必要ひつようになります。 成なり駒こまを作つくっても勝かつまでは長ながいので、ていねいに攻せめる力ちから、そして自陣じじんを守まもりきる力力が必要ひつようになります。

      上手うわてにはいくつかの指さし方かたがあるため、対応たいおうできる柔軟力じゅうなんりょくが求もとめられます。 様々さまざまな形かたちに対応たいおうできるように、棋譜きふ並ならべの学習がくしゅうが重要じゅうようと考かんがえています。 3手詰てづめがスムーズに解とけるようになっていると勝かちやすいです。
