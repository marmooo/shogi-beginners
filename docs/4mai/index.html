四枚落ちのポイント

飛車と角の力を活かして攻めると、右端に弱点があります (左図)。 しかし駒を捨てた攻めになることが多いため、うまく攻めないと反撃も厳しくなります。 受け重視より攻め重視のほうが勝ちやすいですが、 自分の攻めと相手の攻めの速度をよく考えながら指す必要があります。

実戦では王がなかなかつかまらないことが多いです。 そのため大きなミスをしないように注意し、正確な指し回しをするように心掛けましょう。 攻めより受けが得意な子は二歩突き切り (右図) でゆっくり指しても良いと思います。 5手詰ができるようになっていると勝ちやすいです。
