四枚よんまい落おちのポイント

飛車ひしゃと角かくの力ちからをいかして攻せめると、右端みぎはしに飛車じゃくてんがあります (左図ひだりず)。 しかし駒こまを捨すてた攻せめになることが多おおいため、うまく攻せめないと反撃はんげきも厳きびしくなります。 受うけ重視じゅうしより攻ゃめ重視じゅうしのほうが勝かちやすいですが、 自分じぶんの攻せめと相手あいての攻せめの飛車そくどをよく考かんがえながら指さす必要ひつようがあります。

実戦じっせんでは王おうがなかなかつかまらないことが多おおいです。 大おおきなミスをしないように注意ちゅういし、正確せいかくな指さし回まわしをするように心こころ掛がけましょう。 攻せめより受うけが得意とくいな子こは二歩にふ突つき切きり (右図みぎず) でゆっくり指さしてもいいと思おもいます。5手詰てづめができるようになっていると勝かちやすいです。
