八枚はちまい落おちのポイント

十枚じゅうまい落おちのようにすぐに馬うまを作つくることはできません。 しかし守まもり駒こまは少すくないので、上手陣うわてじんを攻略こうりゃくするわかりやすい方法ほうほうがいくつかあります。 まずは左端ひだりはし攻せめ (左図ひだりず)、棒銀ぼうぎん (右図みぎず) をおぼえるのがわかりやすいです。

勝かつためには詰将棋つめしょうぎのように王おうを詰つます必要ひつようがあります。 成なった駒こまをいかしてゆっくり攻せめる技術ぎじゅつ、攻せめ駒ごまを増ふやす技術ぎじゅつ、そしてその間あいだに上手うわての攻せめを受うける力ちからが必要ひつようになります。 3手詰てづめができるようになっていると勝かちやすいです。

詰将棋つめしょうぎのおすすめ書籍しょせき

3手詰てづめの練習れんしゅうをおすすめします。 1手詰てづめは解とけるのに3手詰てづめが解とけなくなる子こは、 2手詰 ~相手の手を読む新感覚トレーニング~ (マイナビ将棋文庫) を使つかった練習れんしゅうをおすすめします。
