ミニ将棋しょうぎのポイント

将棋しょうぎのルールをおぼえた後あと 、裸玉らぎょく (左図ひだりず ) をする人ひとは多おお いのですが、時間じかん をかけてもポイントをつかめないことが多おおいです。 著者ちょしゃが将棋しょうぎ を教おしえる時とき には、短みじかい時間じかん でポイントをつかんでもらうことを重視じゅうし しており、そのツールとして様々さまざま なミニ将棋しょうぎを利用りようしています。

ミニ将棋しょうぎとしては右図みぎず が最もっとも有名ゆうめい で、5x5 の空間くうかんで金銀きんぎん だけを使つかって対局たいきょく するものがあります。 このミニ将棋しょうぎは 22級きゅう の子こによく利用りよう するのですが、ルールをおぼえたばかりの子こ にはまだまだハードルが高たかいことが多おお いです。 そのような時ときにどのように教おし えれば良よいか。 効果こうかの高たか いミニ将棋しょうぎを少すこ しだけ紹介しょうかいしたいと思おもいます。
