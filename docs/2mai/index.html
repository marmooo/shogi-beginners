二枚にまい落おちのポイント

上手陣うわてじんには大おお きな弱点じゃくてんがなく、上手うわて の変化球へんかきゅうも多おお いです。攻せめと受う けの総合的そうごうてきな力ちから が必要ひつようとなります。プロ相手あいて に二枚にまい落お ちで勝かてれば初段しょだんはあります。

どのような変化へんかになっても大切たいせつ なことは、飛車ひしゃ角かく の利ききをいかして上手うわて の金銀桂きんぎんけいを釘くぎ 付づけにすることです。そして残のこ りの駒こまだけを相手あいて にするように心こころ掛が けてください。金銀桂きんぎんけいを釘くぎ 付づけにされた上手うわて の攻せめは弱よわ いので、ていねいに受うければ有利ゆうり を拡大かくだいできます。

金銀桂きんぎんけいを釘くぎ 付づけにするわかりやすい指さ し方かたとして、銀多伝ぎんたでん 定跡じょうせき (左図ひだりず ) 、二歩にふ突つ き定跡じょうせき (右図みぎず ) があります。これら2つの形かたち をイメージしながら指さ しましょう。攻せめるときはどちらも５五ごーごー や４四よんよんの守まも りが薄うすいことに注目ちゅうもく し、守まもりの薄うす い部分ぶぶんをうまく攻せ める方法ほうほうを考かんがえましょう。 7手詰てづみができるようになっていると勝か ちやすいです。
