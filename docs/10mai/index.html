十枚じゅうまい落おちのポイント

上手うわて陣じんは弱点じゃくてんだらけです。 ☗７七歩ななななふ〜☗６六角ろくろくかくとすれば左右さゆうどちらかに馬うまを作つくることができます (左図ひだりず)。 駒こまもたくさん手てに入はいりますが、そこからの勝かち方かたが重要じゅうようです。

勝かつためには詰将棋つめしょうぎのように王おうを詰つます必要ひつようがあります。 つまり馬うまのほかに味方みかたがもう少すこし必要ひつようです。 味方みかたを増ふやす構想力こうそうりょく、そしてその間あいだに上手うわての攻せめを受うける力ちからが必要ひつようになります。 たとえば右図みぎずではどのように攻せめるのが良よいでしょうか。
